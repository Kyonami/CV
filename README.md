### [도전과 성장을 멈추지 않는 클라이언트 개발자, 심교남입니다]
초등학교 시절부터 게임 개발에 관심이 많아 파워포인트나 게임메이커로 게임을 만들기 시작했고, 고등학교 시절부터 C언어와 콘솔 게임 개발로 기본기를 다졌으며, 특성화고 졸업 후 곧바로 실무에 뛰어들어 3년간 유니티 엔진 기반의 상용 게임 개발을 경험했습니다. 실무와 이론의 균형을 맞추기 위해 방통대 컴퓨터과학과를 병행하여 학위를 취득했고, 현업 경험 이후에도 멈추지 않고 개인 프로젝트를 하며 클라이언트 개발자로서의 전문성을 다듬고 있습니다. 단순한 기능 구현을 넘어, 효율적인 아키텍처 설계와 팀의 생산성을 높이는 개발자가 되는 것이 저의 목표입니다.

### [핵심 역량: 라이브 서비스 경험과 효율적인 파이프라인 구축]
- 성공적인 라이브 서비스 경험: 2020년 말 출시된 수집형 비주얼 노벨 게임의 클라이언트 파트를 담당하여, 1년 반 이상 안정적인 라이브 서비스를 이끌었습니다. 신규 콘텐츠(이벤트, 시스템)의 빠른 업데이트 주기를 맞추기 위해 클라이언트 구조를 모듈화하고 최적화하는 데 기여했습니다.

- 신규 프로젝트(Project-H) 기반 제작: 기존 라이브 서비스의 경험을 바탕으로, 신규 프로젝트의 초기 클라이언트 아키텍처 설계에 참여하여 프로젝트의 안정적인 출발을 견인했습니다.

### [제 성격]
저는 기술적 호기심이 강하며, 새로운 엔진 기능이나 아키텍처를 도입하여 기존의 문제점을 해결하는 과정을 즐깁니다. 현업에서 발생하는 성능 병목 현상이나 구조적 한계를 마주했을 때, 주먹구구식 해결보다는 근본적인 원인을 찾고 개선하려 노력합니다.
최근에는 3년간의 실무 경험과, 대학 진학을 통해 알고리즘과 자료구조 등 컴퓨터 과학(CS) 기본기를 더욱 탄탄히 다졌습니다. 단순한 기능 구현을 넘어 성능과 확장성을 보장하는 코드 구조를 설계 해 업무 효율 향상에 도움을 드릴 수 있습니다. 또한, 저는 제가 작성한 코드와 구조를 주변 개발자들과 공유하고 코드 리뷰를 주고받으며 함께 성장하는 문화를 지향합니다.

### [제 강점]
저의 가장 큰 강점은 팀의 빈 곳을 찾아 메우는 솔선수범의 자세입니다.
프로젝트를 진행하다 보면 명확히 담당자가 정해지지 않은 R&D 이슈나 번거로운 유지보수 작업이 발생하기 마련입니다. 저는 이러한 업무들을 선뜻 맡아 처리하며 팀의 전체적인 진행 속도를 높이는 데 기여해 왔습니다.
과거에는 책임감이 강해 혼자서 업무를 끌어안으려는 경향이 있었으나, 실무 경험을 통해 협업의 중요성을 깨달았습니다. 이제는 막히는 부분이 있을 때 적극적으로 이슈를 공유하고 동료들과 피드백을 주고받으며, 팀 단위의 효율적인 문제 해결을 추구하고 있습니다.

### Addressable 에셋 로딩 시스템에 대한 이해

현재 라이브 중인 썸머코드에서는 에셋 번들을 활용하고 있었습니다.
클라이언트 리드엔지니어님께서 관리의 용이함을 위해 새 프로젝트는 Addressable 시스템으로 옮기는 작업을 해주셨고, 
이에 따른 코드 시안, 개선 및 오류 수정을 했습니다.

### 유니티 프로파일러를 활용한 최적화 작업
현재 라이브중인 썸머코드를 폴리싱 하던 과정에서 원인을 알수 없는 프레임 드랍 현상이 있었습니다.
이때 Profiler를 활용해 원인이 되는 코드를 파악하고 해소하는 경험을 했었습니다.
또한, 메모리가 적은 저사양 기기에서 크래시가 일어나는 현상이 있었고,
Memory Profiler를 통해 메모리가 과도하게 사용되는 부분과 누수 현상을 발견했습니다.
이를 정리한 뒤 리드엔지니어님께 전달해 드려 원인을 해결했던 경험이 있었습니다.
